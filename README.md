# teleman-node
Polyfills for teleman in node.js

## Usage
```sh
npm i teleman teleman-node
```

```js
// class
const { Teleman } = require('teleman-node');

// singleton instance
const { teleman } = require('teleman-node');
```

Or

```js
require('teleman-node');
const { Teleman, teleman } = require('teleman');
```
